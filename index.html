Mon Thème WordPress

Ce thème WordPress a été développé dans le cadre de ma formation. Il est conçu pour être responsive et personnalisable.

Installation

  1. Téléchargez le thème depuis la section Releases.
  2. Décompressez le fichier téléchargé.
  3. Téléversez le dossier du thème dans le répertoire wp-content/themes de votre installation WordPress.
  4. Activez le thème via le tableau de bord WordPress.

Fonctionnalités

  • Design responsive
  • Options de personnalisation
  • Support des widgets
